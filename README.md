# Flask-REST-API
## REST API used to do CRUD operations with movie data in SQLite database


### Setup
`git clone <repo-url>`
</br>
`pip install -r requirements.txt`
### Running The App
`python main.py`
### Viewing The App
http://127.0.0.1:5000
### Testing The App
Uncomment and try different requests in test.py </br>
Run test.py in new terminal </br>
`python test.py`
